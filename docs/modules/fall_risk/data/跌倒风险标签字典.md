# 跌倒风险标签字典

本文是跌倒风险方向的统一标签字典。所有人工标注、自动导入标注和训练数据转换都必须使用这里的标签；标注员不得临时发明新标签。

## 1. 使用原则

- 标注员只记录视频里能看见的事实，不做医学诊断，不判断老人真实健康状况。
- 每个动作片段优先选择一个最主要的动作标签；确实同时发生时，在备注中补充次要现象。
- 看不清、被遮挡、人物离开画面或无法确认对象时，用 `U01`，不要猜。
- 风险等级标注由经过培训的复核人员或项目负责人确认，不作为第一阶段人工动作标注任务。
- 同一套标签字典服务所有数据集，但每个数据集只使用自己能支持的标签子集。

## 2. 动作级标签

动作级标签写入：

```text
data/annotations/fall_risk/action_labels.jsonl
```

字段建议：

```json
{
  "video_id": "le2i_home_01_video_1",
  "file_path": "data/external/le2i_imvia/raw/FallDataset/Home_01/Videos/video (1).avi",
  "subject_id": "unknown",
  "scene": "home",
  "view": "fixed_camera",
  "action_id": "A01",
  "action_name": "normal_walk",
  "start_time": 0.0,
  "end_time": 3.8,
  "labeler": "A",
  "review_status": "pending",
  "note": "身体稳定，无明显失衡"
}
```

### 2.1 A 类：正常动作

| 编号    | 英文名            | 中文名  | 什么时候用                       | 不要用于               |
| ----- | -------------- | ---- | --------------------------- | ------------------ |
| `A01` | `normal_walk`  | 正常行走 | 人在连续走路，身体基本稳定，无明显踉跄、扶物或异常下沉 | 只是站立、转身、坐下；明显拖步或摇晃 |
| `A02` | `normal_turn`  | 正常转身 | 人完成转身，过程稳定，没有明显停顿、晃动或扶物     | 转身时明显失衡或需要扶物       |
| `A03` | `normal_sit`   | 正常坐下 | 从站立到坐下，动作连续，最终坐稳            | 身体突然下坠、坐下明显失控      |
| `A04` | `normal_stand` | 正常起身 | 从坐姿到站立，能一次站起并站稳             | 起身很慢、多次尝试或起身失败     |

### 2.2 B 类：轻到中度风险动作

| 编号    | 英文名                 | 中文名  | 什么时候用                  | 不要用于                |
| ----- | ------------------- | ---- | ---------------------- | ------------------- |
| `B01` | `slow_walk`         | 缓慢行走 | 明显走得慢，但仍稳定，没有拖脚、扶物或失衡  | 正常速度行走；已出现拖步、小碎步或摇晃 |
| `B02` | `dragging_walk`     | 拖步   | 脚抬不起来，脚尖或脚底像拖着地面移动     | 只是走得慢但脚能正常抬起        |
| `B03` | `shuffling_walk`    | 小碎步  | 步幅很小、步频偏密、移动缓慢         | 正常小步调整姿势或短距离挪动      |
| `B04` | `swaying_walk`      | 行走摇晃 | 走路时身体左右晃，稳定性差，但未明显差点摔倒 | 明显踉跄后恢复，应标 `C03`    |
| `B05` | `unstable_turn`     | 转身不稳 | 转身时明显晃动、停顿、重心不稳        | 正常稳定转身              |
| `B06` | `slow_sit_to_stand` | 起身缓慢 | 能站起来，但动作慢、费力、需要较长时间稳定  | 多次起身失败，应标 `C01`     |

### 2.3 C 类：高风险前兆动作

| 编号    | 英文名                        | 中文名       | 什么时候用                         | 不要用于                              |
| ----- | -------------------------- | --------- | ----------------------------- | --------------------------------- |
| `C01` | `failed_sit_to_stand`      | 起身失败      | 尝试站起但没有成功，或多次尝试后才站起           | 只是起身慢但一次成功，应标 `B06`               |
| `C02` | `wall_support_walk`        | 扶墙/扶物行走   | 行走或移动过程中持续扶墙、桌、椅、扶手等保持平衡      | 坐下或起身时短暂接触家具，需结合 `A03/A04/B06` 备注 |
| `C03` | `stumble_recovery`         | 踉跄后恢复     | 身体明显失衡、脚步错乱或差点摔倒，但最后站稳        | 真正倒地，应标 D 类跌倒                     |
| `C04` | `rapid_support_contact`    | 快速扶物      | 快要失衡时突然扶墙、扶桌、扶椅子恢复平衡          | 正常扶椅坐下或日常拿取物品                     |
| `C05` | `rapid_body_drop_recovery` | 身体快速下沉后恢复 | 身体突然往下掉、膝髋快速弯曲或重心快速下降，但未倒地并恢复 | 正常坐下、下蹲或弯腰拾物                      |

### 2.4 D 类：跌倒和跌倒后状态

| 编号    | 英文名                      | 中文名   | 什么时候用                     | 不要用于        |
| ----- | ------------------------ | ----- | ------------------------- | ----------- |
| `D01` | `forward_fall`           | 向前跌倒  | 身体明显向前倒下，失去支撑并接触地面、床边、椅旁等 | 主动趴下或正常弯腰   |
| `D02` | `lateral_fall`           | 侧向跌倒  | 身体明显向左或向右倒下，失去支撑          | 只是侧身坐下或侧向跨步 |
| `D03` | `backward_fall`          | 向后跌倒  | 身体明显向后倒下，失去支撑             | 正常坐回椅子或床上   |
| `D04` | `long_static_after_fall` | 跌倒后静止 | 跌倒后躺、趴或侧卧在地面等位置，持续不动或无法起身 | 睡觉、休息、主动躺下  |

### 2.5 U 类：无法判断

| 编号    | 英文名               | 中文名  | 什么时候用                           | 备注要求                              |
| ----- | ----------------- | ---- | ------------------------------- | --------------------------------- |
| `U01` | `unable_to_judge` | 无法判断 | 画面太暗、严重遮挡、人物太小、多人对象无法确认、动作只出现一半 | 必须写明原因，例如“遮挡严重”“人物离开画面”“无法确认是否倒地” |

## 3. 事件级标签

事件级标签写入：

```text
data/annotations/fall_risk/event_labels.jsonl
```

事件级标签用于近跌倒、跌倒、长时间静止等时间窗口。它比动作级标签更适合训练事件检测和评估报警时间。

字段建议：

```json
{
  "video_id": "le2i_home_01_video_1",
  "event_type": "fall",
  "start_time": 6.0,
  "end_time": 6.83,
  "start_frame": 144,
  "end_frame": 164,
  "severity": 4,
  "label_source": "le2i_txt",
  "review_status": "auto_imported",
  "note": "官方 txt 跌倒窗口"
}
```

| 事件类型                   | 中文名   | 对应动作证据        | 说明             |
| ---------------------- | ----- | ------------- | -------------- |
| `normal_activity`      | 正常活动  | `A01-A04`     | 正常行走、坐下、起身、转身等 |
| `near_fall`            | 近跌倒   | `C03/C04/C05` | 明显失衡但最终未倒地     |
| `sit_stand_difficulty` | 坐站困难  | `B06/C01`     | 起身缓慢、起身失败或多次尝试 |
| `unstable_turning`     | 转身不稳  | `B05`         | 转身时出现重心不稳      |
| `wall_support`         | 扶墙/扶物 | `C02/C04`     | 持续扶物行走或突然扶物恢复  |
| `rapid_body_drop`      | 快速下沉  | `C05`         | 身体快速下降但未形成跌倒   |
| `sudden_stop_recovery` | 急停恢复  | `C03/C04`     | 突然停步、调整并恢复平衡   |
| `fall`                 | 跌倒    | `D01/D02/D03` | 明确失去支撑并倒下      |
| `long_static`          | 长时间静止 | `D04`         | 跌倒后持续不动或无法起身   |
| `uncertain`            | 不确定事件 | `U01`         | 事件存在可能，但证据不足   |

## 4. 风险级标签

风险级标签写入：

```text
data/annotations/fall_risk/risk_labels.jsonl
```

风险级标签用于训练最终融合模型，不建议在第一轮动作标注中直接填写。

字段建议：

```json
{
  "video_id": "livingroom_p02_C03_side_t01",
  "subject_id": "p02",
  "start_time": 0.0,
  "end_time": 8.4,
  "risk_level": 3,
  "risk_score": 0.82,
  "risk_factors": ["stumble_recovery", "gait_instability"],
  "label_source": "manual_consensus",
  "review_status": "final"
}
```

| 等级  | 中文名  | 可观察证据                  | 建议动作编码            |
| ---:| ---- | ---------------------- | ----------------- |
| `0` | 正常   | 正常行走、正常坐站、无异常停留        | `record_only`     |
| `1` | 轻微风险 | 轻微慢走、短暂停顿、轻微摇晃，但无明显失衡  | `observe`         |
| `2` | 中度风险 | 拖步、小碎步、明显摇晃、起身缓慢、转身不稳  | `remind_user`     |
| `3` | 高风险  | 近跌倒、扶物恢复、起身失败、明显失衡但未倒地 | `notify_guardian` |
| `4` | 紧急风险 | 明确跌倒、跌倒后长时间静止、疑似无法起身   | `emergency_alert` |

## 5. 质量和备注标记

质量标记不替代动作标签。它们用于解释为什么某段标注可信或不可信。

| 标记                       | 中文名    | 什么时候用              |
| ------------------------ | ------ | ------------------ |
| `clear`                  | 画面清楚   | 人体主体清晰可见，动作边界明确    |
| `partial_occlusion`      | 部分遮挡   | 身体局部被遮挡，但主要动作仍可判断  |
| `heavy_occlusion`        | 严重遮挡   | 关键动作被挡住，通常配合 `U01` |
| `low_light`              | 光线不足   | 画面过暗导致判断困难         |
| `off_screen`             | 离开画面   | 人物部分或完全离开画面        |
| `multi_person_uncertain` | 多人对象不清 | 无法确认应该标哪一个人        |

## 6. 时间边界规则

| 场景    | 开始时间                 | 结束时间                |
| ----- | -------------------- | ------------------- |
| 行走    | 第一步明显开始移动            | 停止行走或切换到转身、坐下、跌倒等动作 |
| 坐下    | 身体明显开始向下坐            | 坐稳、身体不再继续下降         |
| 起身    | 身体明显离开座位或开始上升        | 站稳、身体不再明显晃动         |
| 近跌倒   | 失衡、急停、快速下沉或突然扶物开始    | 恢复稳定，或转为真正跌倒        |
| 跌倒    | 身体开始失去支撑并向地面/床边/椅旁倒下 | 身体接触并稳定在倒地/倒卧状态     |
| 跌倒后静止 | 跌倒动作结束后开始静止          | 明显起身、移动或视频结束        |

时间可以先精确到 `0.1s`；如果工具只能按帧标注，则同时保存 `start_frame/end_frame`。

## 7. 常见混淆

| 容易混淆           | 正确判断                             |
| -------------- | -------------------------------- |
| 走得慢 vs 跌倒风险高   | 标注员只标 `B01`，不要直接写高风险             |
| 慢慢坐下 vs 身体快速下沉 | 正常可控坐下标 `A03`；突然下沉后恢复标 `C05`     |
| 近跌倒 vs 跌倒      | 最后站住了是近跌倒；明确倒下并失去支撑是跌倒           |
| 正常扶椅坐下 vs 快速扶物 | 日常坐下借扶不一定是 `C04`；突然失衡后扶物才是 `C04` |
| 画面看不清 vs 没有事件  | 看不清标 `U01`，不能当作正常负样本             |
