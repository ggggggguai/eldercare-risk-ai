# 心理健康风险算法模块

本模块当前只建立算法工程边界，完整研发计划尚未编写。

## 模块定位

目标是识别心理健康风险变化的可观察信号，输出风险预警和人工复核建议，不输出医学诊断。

## 候选特征

| 特征组 | 示例 |
|---|---|
| 活动节律 | 日均活动量下降、房间移动减少、久坐久卧增加 |
| 睡眠节律 | 夜间活动增加、昼夜节律紊乱、凌晨频繁活动 |
| 社交互动 | 通话或交互频率下降，需看数据是否可获得 |
| 日常规律性 | 起床、用餐、出门、活动时间波动变大 |
| 自评问卷 | GDS、PHQ 类量表可作为人工采集或验证参考 |
| 语音/文本 | 语速、情绪倾向、消极表达等可选增强特征 |

## 第一版算法建议

第一版不直接做抑郁、焦虑诊断。建议先做：

```text
行为节律特征
  -> 个人历史基线
  -> 偏离分数
  -> 风险等级
  -> 人工复核建议
```
